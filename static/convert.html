3 Convert

Converts every .wav under a root folder; optionally includes all subfolders. Original WAVs are never deleted. Use Scan to count files before Run. Output mode is independent of single-file step 2.

4 Root folder Example: a Rekordbox or export tree. Scan finds all WAV paths in sorted order so batch offset/limit is stable between runs.

Bulk output Next to each WAV: FLAC beside the source in the same tree. Mirror: recreate subfolders under Settings destination (avoids collisions across BPM folders). Flat: one folder for all outputs; filenames may use Slot - BPM - Artist - Title when the WAV name matches; tags embed from the same parse.

Mirrored mode keeps different BPM/subfolders from overwriting each other. Flat mode renames when the usual filename pattern matches.

Batching Large libraries: use a batch size and offset. Sorted order is the same every run. Skip-if-exists makes re-runs safe. Clear saved progress forgets the stored offset for this root. After a flat-folder run, open Bulk Fix on that folder. Turning off the limit runs everything in one go—use Scan first.
