4 Bulk (folder tree)

Convert every .wav under a root — including in subfolders (e.g. …/Rekordbox-music/Underground/). Original WAVs are not deleted. Use Scan to count first.

Where to write .flac files (independent of step 2). Names use Slot - BPM - Artist - Title from each filename when you pick a single target folder. Tags are written from the same parse.

Mirrored mode keeps different BPM/subfolders from overwriting each other. Flat mode renames to Artist - Title.flac when the WAV name matches the usual pattern.

Large jobs: convert in smaller runs instead of hundreds of files at once. The same sorted order is used every time, so you can step the offset, or leave “skip if FLAC exists” on and re-run. After a flat folder is filled, use Bulk Fix on that folder in batches too.
