Fix Metadata & Inspect

Fix Metadata is the per-file editor: browse audio on the server, search online catalogues, paste URLs, edit fields, embed artwork, and save—optionally renaming the file to Artist - Title. Inspect is read-focused: a full tag table, artwork preview, Rekordbox-friendly FLAC artwork sizing, and deep links by folder.

Fix Metadata — selecting a file

Enter a folder path or use Browse, Choose folder… (server-side navigator), or Default (Settings destination). The app lists supported audio files. Selecting a file loads current tags into the editor and triggers an automatic combined search when it can derive a query from tags or filename.

With focus outside text fields (and the folder modal closed), use ↑ / ↓, Home, and End to move the highlighted row and load that file—navigation targets the selection instead of only scrolling the list. You can also Tab to the file list (role="listbox") for the same keys.

Deep links: /fix?file=/absolute/path/to/track.flac with optional &q=search+terms to override the search query.

Step 2 — Search results

The server queries Apple Music (iTunes Search API), Discogs, Bandcamp (public search page), and SoundCloud, then merges results. Click a row to fetch full metadata for that URL. If the release has multiple tracks, a tracklist appears so you can pick the correct cut.

When combined search returns no hits, the UI shows a fallback panel: choose Apple Music (default), Discogs, Bandcamp, or SoundCloud, refine the query, and click Search. The app shows up to three results from that catalogue only; pick one to fetch metadata.

Fetch from URL

You can always paste a Bandcamp, Discogs, Apple Music, or Spotify link (or other supported URLs) and click Fetch Metadata without using search.

Step 3 — Edit and artwork

Adjust title, artist, album, year, genre, label, catalogue number, and comment. The cover area shows embedded art, a local file you drop or choose, or catalogue art from search. Click the cover to open a full-screen preview.

Artwork image URL is separate from Fetch from URL (metadata source): paste a direct link to a JPEG/PNG/WebP/GIF file. Update artwork embeds only the cover via POST /api/retag-artwork; Save Tags & Artwork applies text tags too, with priority local cover → artwork URL → release art from search when you save.

Step 4 — Save and optional rename

Save Tags & Artwork writes embedded tags and art. Optional: rename on disk to Artist - Title in the same folder, stripping DJ-style filename clutter. If you configured filename suffix retention under Settings (see Reference → Configurable trailing suffixes), those suffixes are removed when building the search query but kept on the new filename (before the extension)—for example Artist - Title_warped.flac when your source ended in _warped. The metadata source URL is stored in the file (Vorbis comment / ID3 TXXX per format) and may appear on Inspect.

Inspect

Inspect is for verification: list files, select one, and read every tag the app exposes, plus embedded artwork. Fix artwork dimensions (FLAC only) can rewrite cover art to dimensions friendly for Rekordbox. Saved metadata URL displays when the file records which page was used for tagging.

File list keyboard behaviour matches Fix Metadata: ↑ / ↓, Home, and End move selection and load the file when you are not typing in the path field or interacting with the folder modal.

Deep link: /inspect?dir=/path/to/folder. The last browsed folder is shared with Fix Metadata in the browser for convenience.

Audio preview bar

On every main tab, a bar at the bottom streams the currently selected audio file (GET /api/stream-audio) when the format is browser-playable. Use Play / Pause explicitly (no autoplay); drag or click the timeline to seek. Volume is remembered in this browser. Video-only sources on Extract do not preview in the audio element.

UI state in the browser

Fix and Inspect remember drafts in localStorage (djmm.pageState) so returning to the tab restores folders and fields where possible. This is not cookie-based and does not sync across browsers. See Reference.
